# YT to Drive

Personal tools to get music from YouTube into Google Drive and, later, sync that library to a computer or USB drive.

## What’s included

- **Android app** (`android-app/`): share a link into the app; conversion and Google Drive upload are being implemented step by step.
- **Desktop sync** (planned): one-way sync from your Drive music folder to a local folder or USB on Windows and macOS.
